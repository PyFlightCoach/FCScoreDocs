# FCScore Documentation

[FCScore](https://www.flightcoach.org/fcscore/) a web application for the automatic judging of F3A, IMAC and IAC aerobatic flights based on recorded flight data. This site [describes its use](Instructions/index.md), documents the [analysis process](Analysis_Process/index.md) and explains how [scores are calculated](Downgrades/index.md). A high level overview of the scoring process was described in the journal article [Automatic Judging for Precision Aerobatics](https://research-information.bris.ac.uk/admin/files/477795448/Automatic_Judging_for_Precision_Aerobatics.pdf) published in the AIAA Journal of Aircraft.
